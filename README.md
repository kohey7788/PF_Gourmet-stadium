# サイト名称
　ぐるめスタジアム

## サイト概要

スタジアムで食べて美味しかったものを紹介する

### サイトテーマ

サッカー好きで"おいしい"を共有する

### テーマを選んだ理由

スタジアム来てる人に試合だけではなく食を楽しんで欲しいから。
また、サッカーは好きだがスタジアムに来たことない人のきっかけになったらいいと思ったから。

### ターゲットユーザ

サッカーは好きだがまだ一度もスタジアムに行ったことない人
スタジアムにはよく行くがスタグルは食べたことない人
そのスタジアムのことをよく知らないアウェーのサポーター

### 主な利用シーン

サッカー観戦前のお腹がすいた時に
スタジアムで食べた物の美味しさを共有したい時に
